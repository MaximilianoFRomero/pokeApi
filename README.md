
## Installation

```bash
$ npm install

# check has cli installed
$ npm i -g @nestjs/cli

# clone file __.env.template__

# rename file to __.env__ 

# fill preset env parameters
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
## Rebuild database
```bash
#Rebuild database
http://localhost:3000/api/seed
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
